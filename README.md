# Tic-Tac-Toe Game 🎮

A simple Tic-Tac-Toe game built with **React** on **CodeSandbox**.

## Features
- 2-player turn-based gameplay
- Winner detection
- Game reset option

## Tech Stack
- ⚛️ React
- 💻 CodeSandbox

## Live Demo
👉 [Play Now](https://codesandbox.io/) *(replace with your sandbox link)*

## How to Use
1. Clone or open the sandbox.
2. Play the game by taking turns.
3. Click "Reset" to play again!
